TypeScript Tutorial for Beginners

What is TypeScript?

TypeScript is a programming language developed by Microsoft. It's a superset of JavaScript, which means it adds new features and syntax to JavaScript while remaining fully compatible with existing JavaScript code.

The main feature TypeScript adds is static typing. This means you can specify the types of variables, function parameters, and return values. This helps catch errors early in the development process and makes code more readable and maintainable.

Setting Up TypeScript in a Project

To use TypeScript, follow these steps:

Step 1: Install TypeScript

If you already have Node.js installed, open your terminal and run: 

npm install -g typescript

Step 2: Create a Project

Create a folder for your project. Inside the folder, open your terminal and run: 

tsc --init

This creates a tsconfig.json file, which defines TypeScript settings for your project.

Step 3: Write TypeScript Code

Create a new file called app.ts. In TypeScript, use the .ts extension. 

let greeting: string = "Hello, TypeScript!";
console.log(greeting);

Step 4: Compile TypeScript to JavaScript

Run the following command to compile your TypeScript code into JavaScript: 

tsc app.ts

This creates a app.js file that can be used in a browser or Node.js.

Basic Types in TypeScript

TypeScript supports several basic types:

  • number: For numeric values (integers and floats)
  • string: For text values
  • boolean: For true/false values
  • any: For variables that can hold any type of value
  • array: For lists of values
  • tuple: For arrays with a fixed number of elements
  • enum: For sets of named constants
  • void: Usually used as the return type of functions that do not return a value
  • null and undefined: For variables with no value assigned

Example:

let age: number = 25;
let name: string = "John";
let isStudent: boolean = true;
let scores: number[] = [85, 90, 92];
let person: [string, number] = ["Alice", 30]; // tuple
enum Color {Red, Green, Blue};
let c: Color = Color.Green;

Interfaces in TypeScript

Interfaces are a powerful way to define the shape of an object in TypeScript. They allow you to specify which properties an object should have and what types those properties should be.

Interfaces can also be used to define the shape of a function's parameters and return value.

Example:

interface Person {
    name: string;
    age: number;
    greet(): string;
}

let user: Person = {
    name: "John",
    age: 30,
    greet: () => `Hello, I'm ${user.name}`
};

Functions in TypeScript

TypeScript allows you to specify the types of function parameters and return values. This helps prevent errors and makes your code more self-documenting.

You can also use optional parameters, default parameters, and rest parameters in TypeScript.

Example:

function add(x: number, y: number): number {
    return x + y;
}

// Arrow function with optional parameter
const greet = (name?: string): string => {
    return `Hello, ${name || "there"}!`;
};

Classes in TypeScript

TypeScript supports object-oriented programming with classes. Classes can have properties, methods, and access modifiers (public, private, protected).

Example:

class Animal {
    private name: string;

    constructor(name: string) {
        this.name = name;
    }

    public makeSound(sound: string): void {
        console.log(`${this.name} says ${sound}`);
    }
}

let dog = new Animal("Rex");
dog.makeSound("Woof!"); // Output: Rex says Woof!

Generics in TypeScript

Generics allow you to write flexible, reusable functions and classes that can work with different types while still providing type safety.

Example:

function identity(arg: T): T {
    return arg;
}

let output1 = identity("myString");
let output2 = identity(100);

Conclusion

TypeScript adds powerful features to JavaScript, making it easier to write and maintain large applications. By learning TypeScript, you can catch errors earlier in the development process, write more readable code, and take advantage of advanced IDE features like improved autocompletion and refactoring tools.
