TypeScript Flashcards

What is TypeScript?

TypeScript is a superset of JavaScript that adds static typing.

Name three basic types in TypeScript

number, string, boolean

What is an interface in TypeScript?

An interface defines the shape of an object, specifying its properties and their types.

How do you specify a function's return type?

Use a colon followed by the type after the function's parameter list. Example: function add(x: number, y: number): number

What are access modifiers in TypeScript classes?

public, private, and protected

What are generics used for in TypeScript?

Generics allow you to write flexible, reusable functions and classes that can work with different types while maintaining type safety.

What are access modifiers in TypeScript classes?

public, private, and protected

What is the purpose of the tsconfig.json file?

The tsconfig.json file is used to define the compiler options for your TypeScript project

What is the purpose of the any type?

The any type is used when you want to allow a variable to hold any type of value, but it can lead to type errors if not used carefully.

How do you define an interface?

You define an interface using the interface keyword, followed by the name of the interface and a set of properties and methods.
